# STEPFlow V1 - Estado actual del sistema

## OBJETIVO PRINCIPAL

Construir un sistema automatizado de captación y conversión de clientes que permita:

1. Obtener contactos desde múltiples fuentes
2. Iniciar conversaciones automáticamente
3. Generar interés real
4. Llevar a una demo personalizada
5. Convertir en clientes pagos
6. Aprender y optimizar continuamente

---

## MODELO DE OPERACIÓN

STEPFlow funciona como un sistema cíclico:

1. Captación de contactos
2. Clasificación por nicho
3. Generación de mensaje
4. Envío de mensaje
5. Registro de interacción
6. Análisis de respuesta
7. Clasificación de lead
8. Siguiente acción (demo / cierre)

---

## DEFINICIÓN DE ÉXITO (V1)

El sistema es considerado funcional cuando:

- Puede enviar mensajes automáticamente
- Puede registrar interacciones
- Puede detectar respuestas
- Puede identificar leads interesados
- Puede llevar a una demo
